# AI-Based-Crop-Disease-Detection-Using-Leaf-Image-Analysis-and-Farmer-Advisory-System


📌 Project Overview

Crop diseases significantly affect agricultural productivity and crop quality. Early and accurate identification of diseases can help farmers take timely preventive measures.

This project proposes an AI-based crop disease detection and farmer advisory system that uses leaf image analysis through Deep Learning techniques. The system identifies crop diseases from uploaded leaf images and provides disease information, preventive measures, and treatment recommendations to support farmers in making better decisions.

---

🎯 Problem Statement

Traditional crop disease identification methods mainly depend on manual inspection by agricultural experts and laboratory testing. These approaches can be time-consuming, expensive, and inaccessible to farmers in remote areas.

---

💡 Proposed Solution

The proposed system uses a Convolutional Neural Network (CNN) model to analyze crop leaf images and detect diseases automatically. A web/mobile-based interface allows farmers to upload leaf images and receive:

- Disease prediction results
- Confidence score of prediction
- Information about the detected disease
- Preventive and treatment recommendations

🏗️ System Workflow

1. Capture or upload a crop leaf image.
2. Preprocess the image using image processing techniques.
3. Provide the image to the trained CNN model.
4. Predict the disease category.
5. Display confidence score and disease information.
6. Provide suitable preventive and treatment suggestions.
